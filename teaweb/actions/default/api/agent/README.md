# Agent API
* `GET /api/agent` - 获取Agent配置
* `GET /api/agent/pull` - 拉取Agent事件（长连接）
* `PUT /api/agent/push` - 推送Agent事件
* `GET /api/agent/upgrade` - 下载新版本
